## Run Locally
Fork this repository

Clone the project

```bash
  git clone https://github.com/your-username/admitpredict.git
```

Go to the project directory

```bash
  cd admitpredict
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Run on browser

```bash
  python app.py
```
